# Forkify

Forkify is an front-end application that searches through Food2Fork's recipe API for quick and easy recipes.

## Deployment

Live version deployed here: https://frederick-chon.github.io/forkify/

## Built With

* HTML5
* CSS3
* JavaScript
* Webpack
* Babel
